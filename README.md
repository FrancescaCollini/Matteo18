# 🎉 Invito di Compleanno - Matteo 18 anni

Benvenuto nel sito ufficiale per l'invito al compleanno di Matteo! Un'esperienza magica nel Paese delle Meraviglie.

## 🎴 Tema
**Alice nel Paese delle Meraviglie** - Tonalità rosso e bordeaux con carte e tè del Cappellaio Matto

## 📅 Dettagli Evento
- **Data**: 5 Giugno 2026
- **Ora**: 20:00
- **Luogo**: Il Casolare del Sole
- **Ospiti**: ~50 persone

## 🌐 Sezioni del Sito

### 1. 🎴 Il Tema
Presentazione del tema Alice nel Paese delle Meraviglie:
- Dress code suggerito
- Atmosfera del tè del Cappellaio Matto
- Giochi e sorprese a tema

### 2. 📍 Location
Informazioni dettagliate su:
- Luogo: Il Casolare del Sole
- Data e ora dell'evento
- Mappa e indicazioni

### 3. ✓ Conferma Presenza
Sezione protetta con codice di accesso (**wonderland18**):
- Form per la conferma della partecipazione
- Raccolta dati invitati
- Esigenze dietetiche
- Numero di accompagnatori

## 🔐 Codice di Accesso
```
wonderland18
```

## 📱 Caratteristiche

✅ Design completamente responsive (mobile-friendly)
✅ Animazioni eleganti a tema
✅ Palette colori rosso/bordeaux professionale
✅ Sezione conferma protetta da codice
✅ Form per raccogliere dati dei partecipanti
✅ Salvataggio locale dei dati (localStorage)
✅ Export dati in CSV per amministratore
✅ Decorazioni tematiche (carte, tè)

## 🛠️ File del Progetto

- **index.html** - Struttura HTML del sito
- **style.css** - Stili CSS (responsive design, animazioni, tema colori)
- **script.js** - Logica JavaScript (gestione tab, validazione form, accesso protetto)
- **README.md** - Questo file

## 🚀 Come Utilizzare

1. Apri `index.html` in un browser web
2. Naviga tra le tre sezioni usando i pulsanti di navigazione
3. Accedi alla sezione "Conferma" inserendo il codice: **wonderland18**
4. Compila il form di conferma con i tuoi dati
5. Invia la conferma!

## 💾 Dati Salvati

I dati delle conferme vengono salvati automaticamente nel localStorage del browser:
- Nome e cognome
- Email
- Telefono
- Numero di persone
- Esigenze dietetiche
- Data e ora della conferma

### Funzioni JavaScript Disponibili (Console Browser)

```javascript
// Visualizzare tutte le conferme raccolte
getConferme()

// Contare il numero totale di persone confermato
contaPersone()

// Esportare dati in CSV
exportConfermeDati()
```

## 🔧 Personalizzazioni Possibili

Puoi facilmente personalizzare:
1. **Colori**: Modifica le variabili CSS in `style.css` (sezione `:root`)
2. **Testo**: Edita i contenuti in `index.html`
3. **Codice di accesso**: Cambia `ACCESS_CODE` in `script.js`
4. **Data e ora**: Aggiorna i dettagli nella sezione Location
